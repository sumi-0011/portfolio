### es6 class 문법 사용
`React`를 참고한 컴포넌트 기반 구조로 작성하다보니, function 키워드를 사용하면 `setState`와 같은 로직이 중복되는 문제가 발생하였습니다. es6 class 문법의 상속을 이용하여 기본 컴포넌트를 만들고, 상속하여 컴포넌트를 재활용하여 중복 코드를 줄일수 있었습니다.
